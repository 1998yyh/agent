## 总体设计

### 项目目标

**DevAgent** 是一个用于学习的、功能完备的通用软件开发多 Agent 系统。它实现了从**需求分析 → 技术方案 → AI 编码 → 测试生成 → 交付审查**的完整开发流程。

### 设计原则

| 原则 | 说明 |
|------|------|
| **渐进式演进** | 三阶段逐步完善，每阶段都有可用成果 |
| **职责分离** | 每个 Agent 单一职责，易于理解和测试 |
| **上下文传递** | 共享状态管理，确保信息流转 |
| **可观察性** | 完整执行追踪，便于调试和学习 |
| **复用优先** | 基于 claude-quickstarts，避免重复造轮子 |

### 核心组件

| 组件 | 说明 | 实现阶段 |
|------|------|----------|
| WorkflowEngine | 工作流引擎，负责编排和调度 | 阶段 2 |
| ContextStore | 上下文存储，管理状态持久化 | 阶段 2 |
| BaseAgent | Agent 基类，定义通用接口 | 阶段 1 |
| Tool System | 工具系统，提供执行能力 | 阶段 1 |

### 数据流

```
用户请求 → WorkflowEngine → 需求 Agent → 方案 Agent → 编码 Agent → 测试 Agent → 交付 Agent → 交付物
```

## 快速开始

```bash
# 1. 克隆项目
git clone <repository>
cd dev-agent

# 2. 安装依赖
pip install -r requirements.txt

# 3. 设置 API 密钥
export ANTHROPIC_API_KEY='your-api-key'
export ANTHROPIC_BASE_URL=https://coding.dashscope.aliyuncs.com/apps/anthropic

# 可选：指定模型名称
export ANTHROPIC_MODEL=glm-5

# 4. 运行阶段 1
python cli.py
```
